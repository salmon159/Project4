CategoryDomain Associated With Table: YFS_CATEGORY_DOMAIN
The highest level of classification for an item. Multiple domains can be created by an organization. All categorization is defined under a particular domain.
AttributeTypeRequiredTable.Column
AttributeNamestring (maximum length: 100) yfs_category_domain.attribute_name
A pre-defined list of fields that can be used for Item Classification. If AttributeName is passed, it must be a valid attribute name of an item that the system allows. The valid values are: HarmonizedCode, Schedule_B_Code, NMFCCode, NMFCClass, ECCNNo, NAICSCode, UNSPSC, HazmatClass, ProductLine, VelocityCode, TaxProductCode, ItemType, StorageType, PickingType, CommodityCode, and extended attributes of Item.

Validation ConditionError
If IsClassification is set to Y , AttributeName must be passed. If not passed:
YCM0002Mandatory Parameters for the Operation are missing
If passed, AttributeName must be a valid attribute name of an item that the system allows. If not valid:
YCM0050Invalid Item Classification Attribute
