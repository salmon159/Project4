Statestring (maximum length: 35) yfs_reprocess_error_vw.state
The filter criteria is passed in the input XML via the State attribute. When the State attribute is passed as:
    Initial - errors that are in "Initial" state are returned.
    Ignored - errors that are completely in "Ignored" state are returned.
    Saved - errors that have been Saved or Reprocessed one or more times are returned. When a error is saved, a new error is created with "Saved" state. The state of the original error changes to "Modified" state.
    PendingReprocess - errors that are submitted for Reprocessing but not yet processed are returned.
    Fixed - errors that are completely in "Fixed" state are returned.
