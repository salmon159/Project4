PricelistLineAttr Associated With Table: YPM_PRICELIST_LINE_ATTR_VALUE
Supports item attribute based pricing. If the attributes of an item, either in the catalog or overridden for the pricing call, match the attribute conditions for a group, the list price from this table is considered as a valid price for the item.
