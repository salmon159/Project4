OrderLine Associated With Table: YFS_ORDER_LINE
For existing lines, changeRelease can only modify the value of StatusQuantity. Other modifications are ignored by the API.
AttributeTypeRequiredTable.Column
Actionstring.
Specifies the action to be taken.
Valid values:
CREATE - Line is added to the release. The release attributes cannot be specified for this line. The release attributes (see list below) are defaulted from the release. Refer to createOrder for other validations. Bundle cannot be added using this API. It is an error condition to add bundle to a release using changeRelease API. Use changeOrder API to add a new order line for bundle item.
CANCEL - Cancels released quantity from the release line and the corresponding order line. If StatusQuantity and ChangeInQuantity are not provided, the total released quantity is canceled. If StatusQuantity is passed, then the remaining quantity is canceled.
BACKORDER - Backorders released quantity from the release line. The quantity is not changed on the corresponding order line. If neither StatusQuantity nor ChangeInQuantity are provided, the total released quantity is backordered. Partial backordering of Product lines is also allowed.

If: StatusQuantity is passed, and is less than existing StatusQuantity
or ChangeInQuantity is a negative number
the quantity can be either canceled or backordered, depending on the value which is passed for Action. If Action is not specified with either those conditions, the default action is BACKORDER .

If:StatusQuantity is passed and is greater than the StatusQuantity for the OrderReleaseLine
or ChangeInQuantity is positive,
the difference in quantity is added to the release line and to the order line, regardless of the value passed for Action.

Release attributes that are defaulted during CREATE : ShipNode, ReceivingNode, SCAC, MergeNode, CarrierServiceCode, CarrierAccountNo, DeliveryCode, FreightTerms, PackListType, PersonInfoShipTo, CostCurrency, ReqShipDate, ReqDeliveryDate, and References.

Validation ConditionError
Order line with KitCode as BUNDLE is passed
OMP79_0021Bundle cannot be added to a release
Item in the catalog is defined as BUNDLE
OMP79_0021Bundle cannot be added to a release
