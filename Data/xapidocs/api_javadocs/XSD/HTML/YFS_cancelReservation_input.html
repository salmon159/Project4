ReservationIDstring (maximum length: 100) Yesyfs_inventory_reservation.reservation_id
Identification of a reservation.

With Inventory Visibility (IV) integration enabled
This ReservationId can be either of the following
1. Reservation Id of the IV reservation.
In this case, the reservation record corresponding to the given IV reservation Id will be fetched. The other input parameters for this api will not be used to match the reservation record obtained from IV because IV reservation Id can be used to uniquely identify a reservation record in IV. If a reservation record is found, it will be cancelled by calling IV reservation API.
2. Reservation reference of the IV reservation.
In this case, the reservation record(s) corresponding to the given IV reservation reference will be fetched. There may be multiple reservation records corresponding to a given IV reference hence the input attributes ItemId, ShipNode, ProductClass, UnitOfMeasure, Segment and SegmentType will be used to match against the IV reservation records. The matched reservation records will be cancelled by calling IV reservation API.

Validation ConditionError
Must be an existing ReservationId.
YFS10262Blank reservation id.
