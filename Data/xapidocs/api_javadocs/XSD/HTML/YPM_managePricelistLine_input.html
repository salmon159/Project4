PricelistLine Associated With Table: YPM_PRICELIST_LINE
Specify the price list line information to be created, modified or deleted. Either PricelistLineKey or a combination of PricelistHeaderKey, ItemID, UnitOfMeasure, StartDateActive and EndDateActive can be specified to identify a price list line. FromQuantity can be passed along with other attributes to identity a quantity tier price list line only when Operation is Delete .

Validation ConditionError
A quantity tier price list line is identified for the passed input, and the Operation is not Delete .
YPM10021Cannot create or modify quantity tier line without the primary price list line.
Any passed PriceTypeName must exist for the specified Organization.
YPM10191Cannot pass a PriceType that does not exist for the specified Organization.
Any passed PriceTypeName must be associated to the specified Item for the specified Organization.
YPM10194Cannot pass a PriceType that is not associated to the specified Item for the specified Organization.
