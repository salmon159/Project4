Stores additional date attributes for the shipment. DateTypeId must be passed.
AdditionalDate is used to override dates that were calculated for the Shipment, and occur in the Shipment node. For example, by setting the DateTypeId to SCHEDULE , and passing a value for RequestedDate, the value of RequestedShipmentDate for the Shipment is ignored, and the value in AdditionalDate is used.
