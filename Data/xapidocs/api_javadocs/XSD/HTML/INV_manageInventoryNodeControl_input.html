InventoryNodeControl Associated With Table: YFS_INVENTORY_NODE_CONTROL
A combination of OrganizationCode, Node, ItemID, UOM, and ProductClass can be used to indentify an inventory node control. If the inventory node control specified does not exist, it will be created. If it exists, it will be modified.

NodeControlTypestring (maximum length: 20) yfs_inventory_node_control.node_control_type
The type of inventory node control represented by this record. Valid values include:
  • ON_HOLD - Inventory is on hold for sourcing for this item/node combination.
  • IDLE_ASSET - Inventory is idle for this item/node combination.
If not passed, ON_HOLD is assumed.

Validation ConditionError
The value passed for NodeControlType is not a valid value.
INV77_001Invalid node control type.
NodeControlType is either optional or accept ON_HOLD when {AddControlForNode} or {RemoveControlForNode} flag is provided.
INV95_005Invalid node control type.

Allowed ValueDescription
ON_HOLD Inventory is on hold. 
IDLE_ASSET Inventory is idle.
