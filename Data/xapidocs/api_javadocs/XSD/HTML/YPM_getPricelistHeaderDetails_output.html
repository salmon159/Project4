PricelistOrderAttribute Associated With Table: YPM_PRICELIST_ORDER_ATTR
This table maintains the list of Order Attribute filter conditions for Price Lists. Each GroupId represents a set of co-requirements, or ANDed conditions for a price list to be applicable. Only one GroupId condition set needs to be satisfied for a price list to be applicable. A condition can be one of one of three types: text, numeric, or Boolean. For numeric conditions, the condition can either be an exact match, or greater than or equal to a value and less than another value.
