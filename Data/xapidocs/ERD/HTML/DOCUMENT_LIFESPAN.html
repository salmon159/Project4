DOCUMENT_LIFESPAN
( TRANSACTION Table )
This table contains additional lifespan information for a given document. This is commonly used in subsystems like mailbox. Mailbox uses this to add long lifespans to documents until they are used in BPs. This is done so that they are not identified as unassociated orphans that need to be cleaned up by AssociateBPsToDoc.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
