Entity Definition
YFS_INDEX_SYNC
( STATISTICS Table )
This table contains the information whether the index is in sync with the database. The search operation checks this table and searches on index only if it is sync. The index update and sync agents modifies this table with the sync results.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
