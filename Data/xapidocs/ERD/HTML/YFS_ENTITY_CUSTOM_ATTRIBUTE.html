Entity Definition
YFS_ENTITY_CUSTOM_ATTRIBUTE
( CONFIGURATION Table )
This table stores the extended attributes to be used by an Entity.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

ATTRIBUTE_GROUP_IDChar (40)The unique identifier of the attribute group.

  • YFS_ATTRIBUTE (in combination with the ORGANIZATION_CODE, ATTRIBUTE_ID and ATTRIBUTE_DOMAIN_ID columns) 
ATTRIBUTE_IDVarchar (40)The unique identifier of the attribute. This attribute will be used as an extended column for the TableName/DocumentType defined in the corresponding YFS_ENTITY_ATTRIBUTE_DOMAIN table.
A logical foreign key to the following table:
  • YFS_ATTRIBUTE (in combination with the ORGANIZATION_CODE, ATTRIBUTE_GROUP_ID and ATTRIBUTE_DOMAIN_ID columns) 
COLUMN_NAMEVarchar (40)This points to the Column in the hang-off of the corresponding Entity for storing the extended attribute defined in ATTRIBUTE_ID field. 
STATUSVarchar (40)This represents the Status of the extended attribute. The valid values are:
  • PENDING: Indicates that the extended Attribute is mapped to a column in the hang-off of the Entity in TableName for the document type in DocumentType column.
  • ACTIVE: Indicates that the attribute can be used for storing transaction data.
  • INACTIVE: Indicates that the mapped column is not currently used.
  • DELETED: Indicates that the corresponding Column can now be re-mapped to another Attribute. 
SEARCHABLEChar (1)This flag indicates whether the extended attribute is an indexed column in the hang-off. Default value: "N".

CREATEUSERIDVarchar (40)User who created the record. 
MODIFYUSERIDVarchar (40)User who modified the record. 
CREATEPROGIDVarchar (40)Program that created the record. 
MODIFYPROGIDVarchar (40)Program that modified the record. 
LOCKIDNumber (5,0)An integer used to track and prevent concurrent modifications.
