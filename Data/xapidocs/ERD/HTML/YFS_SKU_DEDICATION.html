Entity Definition
YFS_SKU_DEDICATION
( MASTER Table )
This contains definitions of locations that are dedicated to certain SKUs.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription

A logical foreign key to the following table:

A logical foreign key to the following tables:

ITEM_IDChar (40)This field specifies the item's ID. 
UOMVarchar (40)The item's unit of measure. 
PRODUCT_CLASSChar (10)This field specifies the item's product class. 
SEGMENT_TYPEVarchar (40)This field stores customization details for items that are custom specific. 
SEGMENTVarchar (40)This field stores customization details for those items that are custom specific. When the inventory is customized for a specific order, it needs to be tracked separately so that it can be allocated to that order. 
MIN_QTYNumber (14,4)If the inventory at the dedicated location goes below this quantity then replenishment is triggered for the location. Replenishment is triggered by Release Wave and Location Monitor transactions upon additional configuration. 
MAX_QTYNumber (14,4)This quantity indicates the maximum quantity upto which location can be replenished. This quantity is used by Release Wave and Location Monitor transactions while replenishing.
