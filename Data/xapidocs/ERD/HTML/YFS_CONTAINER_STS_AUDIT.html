Entity Definition
YFS_CONTAINER_STS_AUDIT
( TRANSACTION Table )
This table stores container status change audit information. Each time the container status changes because of a business process, an audit is posted along with the status, timestamp, reason code and text, user ID and program ID. All transactions in the container pipeline in fact will have entries in this table upon completion of a transaction.

Records purged from this table are archived into YFS_CONTAINER_STS_AUDIT_H

You can extend this table by adding columns but not by creating hang-off table(s). Please refer to the Customization Guide for detailed procedures and restrictions.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
