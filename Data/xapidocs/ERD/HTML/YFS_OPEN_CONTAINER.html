Entity Definition
YFS_OPEN_CONTAINER
( TRANSACTION Table )
This table stores information about all open pallets available in the system.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription
OPEN_CONTAINER_KEYChar (24)The primary key for the YFS_OPEN_CONTAINER table.  
LPN_NOVarchar (40)The case or pallet License Plate Number. This field is a foreign key to the YFS_LPN_HDR table. 
LPN_TYPEVarchar (40)This field indicates the system defined values for the LPN type associated with the LPN number for which an open container exists. Currently, the system supports only CASE and PALLET LPN types. The valid values are:
  • CASE: Case
  • PALLET: Pallet 
IS_SHIPPING_CONTAINERChar (1)This flag indicates whether the container is a shipping container or inventory container. 
NODE_KEYChar (24)The node key that is associated with the open container.
A logical foreign key to the following table:
  • YFS_SHIP_NODE 
LOCATION_IDVarchar (40)This field indicates the location in a warehouse where the container exists. 
ZONE_IDVarchar (40)This field identifies the zone in which the container exists. 
SHIPMENT_KEYChar (24)The shipment key that is associated with the open container. 
LOAD_KEYChar (24)The load key that is associated with the open container. 
SCACChar (24)The Standard Carrier Alpha Code (SCAC) associated with the open container. This code is assigned to identify the carrier. 
SHIP_DATEDateTimeThe date by which the container must be shipped. 
TARGET_LOCATION_IDVarchar (40)Location to which the container should move the Inventory. 
IS_DUMMY_CONTAINERChar (1)This flag indicates whether the container is a dummy container or not. 
CREATETSDateTimeThe timestamp for the creation of this record. 
MODIFYTSDateTimeThe timestamp for the last modification of this record. 
CREATEUSERIDVarchar (40)The user who created this record. 
MODIFYUSERIDVarchar (40)The user who last modified this record. 
CREATEPROGIDVarchar (40)The program that created this record. 
MODIFYPROGIDVarchar (40)The program that modified this record. 
LOCKIDNumber (5,0)An integer used to track and prevent concurrent modifications.
