Entity Definition
YFS_ATTRIBUTE_REFERENCE
( CONFIGURATION Table )
This table stores references of predefined attribute definition to fields of YFS_ITEM or item alias name.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription
ATTRIBUTE_REFERENCE_KEYChar (24)The primary key for the YFS_ATTRIBUTE_REFERENCE table.

A logical foreign key to the following table:

TABLE_NAMEVarchar (40)The name of the database table that contains the field assigned to a predefined attribute definition. Valid values : 'YFS_ITEM' or 'YFS_ITEM_ALIAS'. 
REFERENCE_NAMEVarchar (40)The reference name that the predefined attribute represents. If the tablename is 'YFS_ITEM', then this field contains the XML name of the field that the attribute represents. If the tablename is 'YFS_ITEM_ALIAS', then this field contains the item alias name that the attribute represents.
