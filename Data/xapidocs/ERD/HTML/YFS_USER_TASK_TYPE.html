Entity Definition
YFS_USER_TASK_TYPE
( CONFIGURATION Table )
This table stores task types that a user group performs. System should suggest only those task to the user that belong to one of the task types specified here. If no task types have been specified, it means that any type of task can be suggested to the user.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
