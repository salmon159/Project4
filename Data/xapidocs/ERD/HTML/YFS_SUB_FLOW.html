Entity Definition
YFS_SUB_FLOW
( CONFIGURATION Table )
This table stores the configuration information at the sub flow (run time) level. Every service that is configured is translated to one or more sub flows. Depending on configuration, of each sub flow can be invoked synchronously or asynchronously by starting the IntegrationServer daemon).



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
