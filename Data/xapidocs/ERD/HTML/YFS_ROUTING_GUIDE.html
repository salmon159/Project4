Entity Definition
YFS_ROUTING_GUIDE
( CONFIGURATION Table )
This table stores Routing Guide Header Information.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription
ROUTING_GUIDE_KEYChar (24)The primary key for the YFS_ROUTING_GUIDE table.  
ROUTING_GUIDE_NAMEVarchar (100)Name of the Routing Guide. 
ORGANIZATION_CODEChar (24)The organization for whom this routing guide is applicable.
A logical foreign key to the following tables:
  • YFS_FREIGHT_TERMS (in combination with the FREIGHT_TERMS and TYPE columns)
  • YFS_ORGANIZATION 
TYPEChar (24)Typically holds the values INBOUND or OUTBOUND and reflects the type of the routing guide based on the organization (buyer, enterprise or hub) owning the routing guide.
A logical foreign key to the following table:
  • YFS_FREIGHT_TERMS (in combination with the ORGANIZATION_CODE and FREIGHT_TERMS columns) 
DEPARTMENT_CODEVarchar (20)Department that deals with this item. 
ITEM_CLASSIFICATIONVarchar (40)A classification that distinguishes the various merchandise that are dealt by the retailer who defined this routing guide. 
FREIGHT_TERMSChar (24)Freight Terms for this Routing guide.Can be Collect, Prepaid or 3rd Party.
A logical foreign key to the following table:
  • YFS_FREIGHT_TERMS (in combination with the ORGANIZATION_CODE and TYPE columns) 
ROUTING_GUIDE_NOVarchar (40)A number that identifies this routing guide entry. 
SUPERCEDES_ROUTING_GUIDE_NOVarchar (200)The routing guide number that preceded the current routing guide number.
