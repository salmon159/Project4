Entity Definition
YFS_HOLD_TYPE_MOD_TYPE
( CONFIGURATION Table )
This table stores the mapping of the hold types that should get enabled when a modification occurs on a document type, as well as the mapping in between the hold types and the prevented modifications types.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
