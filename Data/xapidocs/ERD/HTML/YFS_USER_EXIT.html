Entity Definition
YFS_USER_EXIT
( CONFIGURATION Table )
This table stores all the user exits.



You can extend this table by adding columns but not by creating hang-off table(s). Please refer to the Customization Guide for detailed procedures and restrictions.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription
USER_EXIT_KEYChar (24)The primary key for the YFS_USER_EXIT table.  
USER_EXIT_NAMEVarchar (255)The name of the user exit. 
ORG_OVERRIDE_FLAGChar (1)This flag indicates whether the user exit can be implemented by the organization. This field is currently not supported. 
DOC_TYPE_OVERRIDE_FLAGChar (1)This flag indicates whether this user exit can be overriden for a given document type. For example, the document type can be a sales order or a return order. 
PROPERTY_NAMEVarchar (255)This field is used internally. 
FLOW_DESIGNABLEChar (1)This flag indicates whether the user exit can invoke a service directly. 
BASE_DOCUMENT_TYPE_KEYChar (24)The base document type for the user exit. This field is currently not supported. 
PROXY_CLASSVarchar (128)The name of the proxy class.
