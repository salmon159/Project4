Entity Definition
YFS_ORDER_SEARCH_CONDITION
( TRANSACTION Table )
This table stores Order search conditions associated to a Return Order. For CustomAttributes, the EnterpriseCode, DocumentType, EntityAttributeName, and EntityAttributeValue attributes are required. For extended columns of YFS_ORDER_HEADER, EntityAttributeName and EntityAttributeValue are required.

Records purged from this table are archived into YFS_ORDER_SEARCH_CONDITION_H

This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
