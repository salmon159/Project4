Entity Definition
YFS_ORDER_LINE_PRICE_MATCH
( TRANSACTION Table )
This Table is a hang off table for the yfs_order_line table and stores the information about the price match and the relationship between Order Line and Price Match

Records purged from this table are archived into YFS_ORDER_LINE_PRICE_MATCH_H

This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
