Entity Definition
YFS_DISTINCT_ORDER_LINE_VW
( TRANSACTION Table )
The view formed by joining the YFS_ORDER_HEADER and YFS_ORDER_LINE. This view represents the some of the common attributes between the two tables and updates the attributes in YFS_ORDER_LINE having null values to the values present in YFS_ORDER_HEADER.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
