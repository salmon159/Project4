Entity Definition
YFS_ORDER_LINE_OPTION
( TRANSACTION Table )
This table stores information about Options associated with a Service Order Line.

Records purged from this table are archived into YFS_ORDER_LINE_OPTION_H

You can extend this table by adding columns and by creating hang-off table(s). Please refer to the Customization Guide for detailed procedures and restrictions.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
