Entity Definition
YFS_CAPACITY_AVAILABILITY
( TRANSACTION Table )
This table stores resource pool capacity consumption for each date/shift combination.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

  • PS: Indicates that this resource pool is used for provided service items.
  • SS: Indicates that this resource pool is used for special service items.
  • PROD: Indicates that this resource pool is used for product items. 
PURPOSEVarchar (10)This field indicates if the resource pool is used for service or inventory scheduling. The valid values are:
  • SERVICE: Indicates that the resource pool is for delivery or provided services scheduling.

  • DOCK: Indicates that the resource pool is for dock scheduling.
  • WMS-RP: WMS Resource Planning
  • INVENTORY: Indicates that the resource pool availability should be consider when scheduling products.
