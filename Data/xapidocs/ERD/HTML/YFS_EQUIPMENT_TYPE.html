Entity Definition
YFS_EQUIPMENT_TYPE
( CONFIGURATION Table )
All equipment types of the warehouse are defined in this table. e.g. GRABTRUCK, FORKLIFT etc.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

MODELED_ASVarchar (40)This field indicates whether the equipment type is modeled as Case, Pallet or Location. The three supported values are 'Case', 'Pallet', and 'Location'. The valid values are:

  • PALLET: Pallet

IS_STATIONChar (1)This flag indicates whether the equipment type is modeled as a station. 
EQUIPMENT_LOCATION_TYPEVarchar (40)This field is a system defined classification of location to aid association of certain types locations for certain other operations with WMS. The supported types are: INTRANSIT (Mobile locations), STAGING, VIRTUAL, REGULAR, DOCK. For example, all mobile equipment locations should be of type INTRANSIT. All Cart locations should be of type DROP-OFF or REGULAR, based on whether the location is the final drop location for the picking operation or not. The valid values are:
  • INTRANSIT: In Transit Location
  • STAGING: Staging Location
  • VIRTUAL: Virtual Location
  • REGULAR: Regular Location
  • DOCK: Dock Location
