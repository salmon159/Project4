Entity Definition
YPM_PRICELIST_LINE_ATTR
( MASTER Table )
Supports range(through virtual columns)/absolute(decimal attr1..5, string 1...5) match based pricing.Virtual Columns support range based pricing through attributes RangeAttr1...5, ie if RangeAttr1's value captured, say falls between MinAttr1(LTE) and MaxAttr1(GT), list price from this table is fetched rather from PricelistLine Table. Relationship with PricelistLine table is oneToMany in nature



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
