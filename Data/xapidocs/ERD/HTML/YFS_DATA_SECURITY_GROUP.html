Entity Definition
YFS_DATA_SECURITY_GROUP
( CONFIGURATION Table )
Data Security groups that are used to control end-user access to the application data elements are stored here.



You can extend this table by adding columns but not by creating hang-off table(s). Please refer to the Customization Guide for detailed procedures and restrictions.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

CUSTOMER_ACCESS_MODEChar (2)Access mode for Customer. Indicates if the team can access all customer data (01), only specified customers (02) assigned to the team, or only specified customers assigned directly to the user, regardless of the Enterprise's manual assignment rule (03). Default Value is 01. The valid values are:
  • 01: Normal access mode. The team can access all customer data.
  • 02: Restricted based on team. Access to customer data is restricted based on team assignments.
  • 03: Restricted based on user. Access to customer data is restricted based on direct customer assignments regardless of the Enterprise's manual assignment rule.
