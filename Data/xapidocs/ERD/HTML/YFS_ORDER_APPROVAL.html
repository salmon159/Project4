Entity Definition
YFS_ORDER_APPROVAL
( TRANSACTION Table )
This table stores order approval records that are created while evaluating whether a quote needs approvals. These records are created based on order approval rules and approval plan applicable to the quote. The UserGroupID and TeamCode determine the user who can approval an order approval record.

Records purged from this table are archived into YFS_ORDER_APPROVAL_H

This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
